Vertical rhythm

Line height measures everything.

The main idea is to make a grid of rows with 16px height of each.

Height of everything on the page should be divisible by 16.

This rule is applied to margins and total height of each element.

Line-heights are discrete: 16px, 32px, 64px.

Line height is NOT a font size multiplied by some value like 1.5 as usual.
There are 3 groups of possible font sizes suitable for selected line heights:

16px: 12 and 14 px fonts
32px: 16–32 px fonts
64px: 40–64 px fonts

Why not 48px?

16px is the minimal possible line height. It is acceptable for 14px font size and less.
I also don't recomment do use a size less than 12px.
So, 16px is the most commonly used font size and 32px line height makes it more readable. A good pair.
48px = 32px * 1.5. It means that half of lines are not "in rhythm".
